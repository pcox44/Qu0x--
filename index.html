Game #
  • Use all five dice values exactly once per expression by clicking the dice.
  • Build expressions using the buttons below to reach target numbers 1 through 100 each week.
  • Allowed operations: + - * / ^ ! (factorial, supports single factorial only).
  • Parentheses are supported.
  • Backspace removes one character from expression and restores dice if number removed.
  • Clear resets the expression and dice.
  • Submit to check your expression and cross off any numbers you made this week.
  • Get all 100 to achieve a Qu0x 100!
